О проекте

АкылСөз — GovTech-ориентированный проект, который повышает качество кыргызоязычного контента и расширяет тематику: от бытового контента → к академическому (AI, IT, наука). Мы используем нейросети, чтобы делать современную картинку и чистую озвучку при минимальных затратах.

AkylSöz = (Generative Video) + (Neural Voice) + (A1–C1 Language Bridge) + (STEM Narratives)

1) Качество

Низкое качество визуала и озвучки в кыргызоязычной среде из-за бюджетов и нехватки продакшна.

2) Тематика

Преобладает “быт” и грамматика. Почти нет AI/IT и научных объяснений на кыргызском языке.

3) Барьер языка

Чистый кыргызский сложен для части аудитории. Нужен мягкий вход без стресса.

Решение

  • AI-продакшн
    KlingAI для визуала + Kanitts для озвучки → качество при минимальных затратах.
  • Уровневый языковой мост
    KG+RU (A1→C1): постепенное увеличение кыргызского до почти уровня носителя.
  • Академическая среда
    Детский формат о AI/IT → дальше расширение на весь спектр науки.

Как мы работаем (пайплайн)

1) Сценарий + уровни
Тема → цели → лексика → A1/A2/B1-ограничения → реплики → интерактив
2) Генерация визуала
KlingAI: сцены + стиль персонажей + правки
3) Озвучка
Kanitts: голоса → чистка → тайминг под субтитры
4) Монтаж + субтитры
Подсветка KG-слов • бейдж уровня • темп 30–45 сек / эпизоды 8–12 мин
5) Публикация + метрики
Shorts/TikTok → retention → возвраты → комментарии → итерации формата
6) GovTech пилот
Пилот 30–90 дней → отчёт → апробация → контракт/масштаб

Масштабирование

После пилота в Кыргызстане модель переносится на другие языки и регионы: Узбекистан, Казахстан, Якутия, Татарстан, Турция и др. Пайплайн остаётся, меняются лингвистические правила и локальный контекст.

Локализация

Носители языка + методист → адаптация уровней и словаря.

Партнёрства

Госканалы, EdTech, научпоп-сообщества, диаспора.

Экспорт формата

Лицензирование + совместное производство сезонов.
